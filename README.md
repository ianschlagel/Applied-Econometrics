# Applied Economics: Structural & Financial Modeling

This repository contains research projects focused on applied econometric modeling within financial and international commodity markets. These projects emphasize high-frequency data analysis, time-varying correlations, and micro-consistent structural systems.

---

## 🔬 Featured Projects

### 1. The Duality of Risk and Safety in the Bitcoin Market
**Method:** DCC-GARCH (Dynamic Conditional Correlation)  
**Tools:** Stata, Python (`yfinance`)
* **Overview:** Investigates Bitcoin's context-dependent role as either a **risk-on asset** or a **safe-haven asset** during macroeconomic shocks.
* **Key Innovation:** Utilizes a DCC-GARCH framework to identify Bitcoin as a **"Schelling Point"**—a systemic-risk hedge specifically during banking crises (e.g., the 2023 SVB collapse) compared to general market uncertainty (e.g., Covid-19).
* **Technical Note:** Built for **Universal Reproducibility**.

### 2. Portugal's Wine Demand: An AIDS Framework
**Method:** Almost Ideal Demand System (AIDS)  
**Tools:** R (`micEconAids`, `systemfit`, `tidyverse`)
* **Overview:** An empirical analysis of locational preferences in the Portuguese wine market, analyzing the competitive dynamics between France, Italy, and Spain.
* **Key Innovation:**: Utilize an AIDS model to investigate Portugal's import Demand of the big three European wine producers.
* **Technical Note:** Built for **Universal Reproducibility**; the R script pulls cleaned FAO data directly from this GitHub repository, requiring zero local file configuration.
